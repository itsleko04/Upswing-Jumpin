## 1. Общие сведения
**Upswing Jumpin'** — это динамичный ритм-платформер на языке Python, вдохновленный классическими представителями жанра (такими как Geometry Dash). Проект разработан командой из двух человек в рамках учебного среза знаний. Основная цель — демонстрация навыков объектно-ориентированного программирования, знаний в работе с Python, работы с Arcade.

*   **Николай Липницкий (Lead Programmer)** — архитектура движка игры, логика игрока, система уровней.
